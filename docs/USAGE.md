# Инструкция по использованию бота-помощника

## Основные возможности

### Для пользователей:
1. **Поиск в базе знаний** - поиск информации в загруженных документах
2. **Поиск в интернете** - поиск информации в интернете через DuckDuckGo
3. **Задать вопрос ИИ** - прямой запрос к подключенному ИИ (по умолчанию Ollama)
4. **Обработка изображений** - описание изображений с помощью мультимодальных моделей
5. **Настройки** - выбор провайдера ИИ

### Для администраторов:
1. **Управление пользователями** - одобрение/отклонение/удаление пользователей
2. **Управление базами знаний** - создание, загрузка документов, очистка, удаление
3. **Настройки ИИ** - выбор и настройка провайдеров ИИ
4. **Загрузка документов** - поддержка форматов:
   - Markdown (.md)
   - Word (.docx, .doc)
   - Excel (.xlsx, .xls)
   - PDF (.pdf)
   - Веб-страницы (URL)
   - Изображения (.jpg, .png, .gif)

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Настройка

1. Скопируйте `env.template` в `.env`:
   ```bash
   # Linux / macOS
   cp env.template .env
   
   # Windows PowerShell
   Copy-Item env.template .env
   ```

2. Отредактируйте `.env` и заполните необходимые параметры:
   - `TELEGRAM_BOT_TOKEN` - токен бота от @BotFather (обязательно)
   - `ADMIN_IDS` - список Telegram ID администраторов через запятую (обязательно)
   - `MYSQL_URL` или `DB_PATH` - настройка базы данных (см. [DATABASE.md](DATABASE.md))
   - `OLLAMA_BASE_URL` - URL сервера Ollama (по умолчанию http://localhost:11434)
   - `OLLAMA_MODEL` - название модели Ollama (по умолчанию deepseek-r1:1.5b)
   - `REDIS_HOST` и `REDIS_PORT` - настройки Redis (опционально)

Подробнее о конфигурации см. [CONFIGURATION.md](CONFIGURATION.md)

## Запуск

```bash
python bot.py
```

## Использование

1. Отправьте `/start` боту
2. Дождитесь одобрения администратором
3. Используйте кнопки для навигации по функциям

## Примечания

- Для работы RAG системы рекомендуется установить `sentence-transformers` и `faiss-cpu`
- Для обработки изображений требуется мультимодальная модель в Ollama
- Для OCR (распознавание текста на изображениях) требуется `pytesseract` и Tesseract OCR
